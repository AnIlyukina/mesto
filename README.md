# Проект: *Mesto*
Alt-H1

### Описание проекта
* Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки :)


**Технологии**

Файловая структура организована по БЭМу.
Вёрстка адаптивная
Реализована работа “pop-up" при помощи JavaScript
